쇼핑몰 사이트 제작


해당 주제를 선정한 이유?(목적)

쇼핑몰에 관심이 있어, 수업 중에 배운 HTML과 CSS를 이용해서 싱글 페이지를 제작해보았습니다. 개발기간은 약 2일 소요되었고, 깃허브에서 소스를 확인하실 수 있습니다.

Semantic UI를 기초로 HTML 레이아웃을 구성했고, 수업시간에 배운 Flex와 Grid를 중점으로 CSS를 작성했습니다. 미디어 쿼리를 사용하지 못한 점이 프로젝트의 보완점입니다.
