### CSS 폰트 적용 방법
```
1. https://fonts.google.com/ 접속
2. 폰트 검색
3. 우측 Select this style 클릭해서 폰트 추가하기
4. import 복사 -> css 맨위에 복붙 (css 파일 나눴으면 style태그 빼고 복붙)
* link 복사일 경우엔 html 파일 title위에 복붙하기
5. 적용할 태그 css에 폰트 이름 복붙
    ex) font-family: 'Shizuru', cursive;
```
