## ul, li, ol
    - <ul> = Unorder List(정렬되지 않은)
    - <li> = List Item

    <ul>
        <li>
        <a href="쇼핑몰주소">상의</a>
        </li>
          <li>
        <a href="쇼핑몰주소">하의</a>
        </li>  
       
        <li>하의</li>
        <li>신발</li>
        <li>모자</li>
    </ul>

    - <ol> = Order List (*실무에서 잘 안씀) 번호가 매겨짐

    <ol>
    <li>JAVA</li>
    <li>PYTHON</li>
    <li>MTSQL</li>
    </ol>

## css position
    태그마다 position을 가지고 있음
    position 디폴트값이 static/
    1. static (디폴트) : 위치를 지정하지 않을 때 
    2. relative : 원래 

    static은 돌이라 움직이지 않음   그래서 relative로 설정하고 움직여야함
