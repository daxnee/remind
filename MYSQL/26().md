## 220216
### 모델링 복습
    1. cascade
    : 부모데이터 (삭제or 수정)시 자식 데이터 (삭제or수정)

    2. set null
    : 부모 데이터 (삭제or수정)시 자식 데이터 null

    3. set default 
    : 부모 데이터 (삭제or수정)시 자식 데이터에 default 값

    4. restrict
    : 부모 데이터 (삭제or수정)시 자식 데이터에 (삭제or수정) 불가능

    5. no action
    : 아무런 변화 없음
